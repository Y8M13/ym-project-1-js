# Tasks Manager

### OVERVIEW
Task Manager will be used to create new tasks and keep a record of all tasks that have been loaded.  

### FEATURES
- Home Page (main page)
- Nav Bar (for navigations)
- Task Form (to submit new tasks)
- Submit button
- Task Log (list of tasks)

### MVP
- Home Page Page
- Task Log Page
- Task Form Page
- Navigations
- Submit button

### STRETCH GOALS
- Delete button
- Update button
